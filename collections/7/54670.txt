SQL Server Management Studio Express (SSMSE) is the primary SQL Server Express management and development tool. Using keyboard shortcuts lets you do more work faster, so one of the best ways to be productive with SSMSE is to understand the available keyboard shortcuts. SSMSE supports quite a few keyboard shortcuts, but here are the most common. 1  CTRL+C, CTRL+X, CTRL+V, CTRL+Z, CTRL+Y--Copy, Cut, Paste, Undo, Redo.
