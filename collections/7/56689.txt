To use the command line you must access Command Prompt in Windows. Command Prompt simulates MS-DOS, so you must use various DOS commands when you work in it. To change the drive in command line, you can use the cd command.The Change Directory (cd) command allows you to navigate through different drives. The cd command is used if you want to change to a specific folder location on the drive; otherwise, the drive can be entered at the top level.nstructions. Type cmd at the Start screen and click on Command Prompt when it appears. You will now be able to work in the command line. Type cd /D X:\folder if you want to change to a specific folder in another place.
