Did you know that you can assign more that one IP address to a single physical network interface? This technique is quite useful, for example when working with Apache and virtual hosts, as it allows you to access same Apache server by using two different IP addresses. Temporary virtual network interface. The process of creating a virtual network interface in Linux is a quite simple matter. It involves a single execution of the ifconfig command. The above command will create a new virtual network interface based on original eth0 physical interface.
