How to change the style of the default font in Word 2002 or in Word 2003. 1  If you do not have a document open, create a new document that is based on the Normal template. 2  To do this, click New on the File menu, and then click Blank Document in the New Document task pane. 3  On the Format menu, click Font. In the Font group, click the Font flyout. 2  Select the options that you want to apply to the default font, such as font style and font size. 3  If you selected specific text, the properties of the selected text are set in the dialog box. 4  Click Set As Default.
