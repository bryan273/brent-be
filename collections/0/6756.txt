How to change the style of the default font in Word 2010. 1  If you do not have a document open, create a new document that is based on the Normal template. 2  To do this, click File, click New, click Blank Document, and then click Create.3  In the Font group, click the Font flyout. In the Font group, click the Font flyout. 2  Select the options that you want to apply to the default font, such as font style and font size. 3  If you selected specific text, the properties of the selected text are set in the dialog box. 4  Click Set As Default.
