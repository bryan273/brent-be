When you connect to a new wireless network, Windows will create a profile for the wireless network. A wireless (Wi-Fi) network profile contains the SSID (network name), password key, and security information to be able to connect to a wireless network.This tutorial will show you how to manually set up and connect to a wireless network on your PC in Windows 10.ou can now close the Settings app if you like.
