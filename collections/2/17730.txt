The easiest way to move a Web site or Web application to Azure is to deploy it to Azure Web Sites; very few, if any, changes are required to the application. You can deploy from Microsoft Team Foundation Server (TFS) or other source code repository systems such as GitHub.s you’ll see later in this article, the choice between IaaS and PaaS isn’t limited to application code. When you follow the IaaS route, you can choose to deploy an Azure Virtual Machine (VM) that has a database such as SQL Server or MySQL preinstalled.
