Named instance. When you run SQL Server Express, by default, the named instance option is selected with SQLExpress as the instance name. Choose this option to create a new named instance. Consider the following when you name an instance of SQL Server: 1  Instance names are not case sensitive. 2  Instance names cannot contain the terms Default, MSSQLServer, or other reserved keywords. 3  If a reserved keyword is used in an instance name, an error will occur.
