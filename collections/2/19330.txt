2012 ICD-9-CM Diagnosis Code 374.02. Mechanical entropion. ICD-9-CM 374.02 is a billable medical code that can be used to indicate a diagnosis on a reimbursement claim, however, 374.02 should only be used for claims with a date of service on or before September 30, 2015. For claims with a date of service on or after October 1, 2015, use an equivalent ICD-10-CM code (or codes).
