What is a file geodatabase? A file geodatabase is a collection of files in a folder on disk that can store, query, and manage both spatial and nonspatial data. You create a file geodatabase in ArcGIS. File geodatabases are made up of seven system tables plus user data. 1  User data can be stored in the following types of datasets: 2  Feature class. 3  Feature dataset. 4  Mosaic dataset. 5  Raster catalog. 6  Raster dataset. 7  Schematic dataset. 8  Table (nonspatial) 9  Toolboxes.
