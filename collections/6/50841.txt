The first thing we need to do is kill all hanging Outlook processes. To do this, start up the Windows Task Manager either by pressing Ctrl + Alt + Del, or by right clicking the taskbar and choosing  Start Task Manager . Then switch to the Processes tab and find all OUTLOOK.EXE items in the list.hen open Windows Explorer and navigate to the C:\Program Files\Microsoft Office\OFFICE14 folder if you are using Outlook 2010. If you have Outlook 2013 installed, it will be C:\Program Files\Microsoft Office\OFFICE15. Double-click Scanpst.exe and click  Browse  to select the .pst or .ost file you want to check.
