Defend your computer against viruses, spyware, and other malware. Microsoft Security Essentials is a free download for Windows 7 that helps protect your computer from viruses, spyware, worms, Trojans, and other malware. For more information, see Help protect your PC with Microsoft Security Essentials. Windows 7 also includes Windows Defender, software that helps protect your computer from pop-up ads, slow performance, and security threats caused by spyware and other unwanted software. For more information, see Using Windows Defender. Note: In Windows 8, Windows Defender replaces Microsoft Security Essentials.
